# XuexitongJSFork 学习通一键全自动刷课脚本

## 项目简介

本项目灵感来自 [@unraous](https://github.com/unraous) 的 [uXuexitongJS](https://github.com/unraous/uXuexitongJS)。在原有基础上重写，去除AI 智能答题等功能，适配tampermonkey。

## 功能特性

- 自动识别课程树结构，自动切换未完成章节
- 自动播放视频、自动切换倍速、自动静音
- 自动检测并翻页 PDF 文档
- 自动识别并答题（支持单选、多选、判断题）
- 支持后台播放，节省时间
- 兼容 FireFox、Edge 等主流浏览器
